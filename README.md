# Quote Generator App

A simple web app that fetches and displays **random quotes** from [DummyJSON Quotes API](https://dummyjson.com/quotes) using **HTML, CSS, and JavaScript**.

## Features

- Fetch quotes dynamically from the API  
- Display a **random quote** on button click  
- Smooth **fade-in/fade-out animation**  
- Built with **HTML, CSS, and JavaScript (Fetch API)**  

## Demo

Click the button to generate a random quote.

## Technologies Used

- HTML – Page structure  
- CSS – Styling & animations  
- JavaScript – Fetch API for quotes  
- DummyJSON API – Source of quotes  

## How It Works

1. App fetches quotes from DummyJSON API.  
2. Stores the quotes in a variable.  
3. On button click:
   - Picks a random quote  
   - Applies fade animation  
   - Displays it in the UI
